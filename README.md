# Manual CMA

Projeto pronto para publicação no GitHub Pages.

## Como publicar
1. Envie todos os arquivos deste pacote para o repositório `manual.cma`.
2. No GitHub: Settings > Pages.
3. Em Source, escolha `Deploy from a branch`.
4. Branch: `main` e pasta `/root`.
5. Salve.

O link ficará parecido com:
`https://cmapessoal-dev.github.io/manual.cma/`
